# The Experimental Features

In this folder, you can find some Jupyter Notebooks describing some new experimental features that are available in the recent `ysoisochrone` updates. Some of these features are still under development, and some of them are used in some works that have not been published. A more detailed tutorial on using these features will be released once the method has been tested and/or the related publication is released. If you find these features useful, and want to use this updated feature now, please contact us, and we would be happy to help and collaborate to make what you need possible. 

## Recent Updates:

### v1.1.0

Include the ability to apply prior to the Bayesian method. A common prior could be the stellar masses estimated from the dynamical methods.

### v1.1.1-1.1.3

Include a few more useful stellar evolutionary tracks, including the Feiden magnetic tracks, Siess2000, SPOTS models, and Pisa models (basic information has been added to `documentation/models`, and more information will be added soon).

Have the function to force the code to run through even it finds some observation points are outside of the tracks. This is a dangerous approach but is useful for larger dataset.

Some other user experience updates: now more models can be selected.
