# README for Data Matrix

Here are the collection of prepared matrix for out of box usage. These data matrix serve as a archive of the existing stellar evolutionay tracks. 

If you use any of the matrix we provided here, please still refer to the original publication. Please also refer to the original publication for updated tracks.

If you find the tracks provided here cannot meet your need, please also download the tracks from their original website and also modify the data matrix using the function to build your own matrix (see tutorial for more details).
